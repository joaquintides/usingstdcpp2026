[The Mathematical Mind of a C++ Programmer](https://github.com/joaquintides/usingstdcpp2026/raw/main/The%20mathematical%20mind%20of%20a%20C%2B%2B%20programmer.pdf):
Many people hate math, some programmers do too. We hold that this hate stems from a reductionist
view of mathematics as merely rule application: number crunching, equation solving and the like.
But programming is more akin to mathematical creation, and writing a well designed, neat program
can be as exhilarating as devising a new little math theory. In this talk we'll investigate how
the mathematical mind approaches the world, and how developing a mathematical inclination can 
help you be a better C++ programmer.
